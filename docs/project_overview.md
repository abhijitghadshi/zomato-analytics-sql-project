# Zomato Analytics SQL Project Documentation

# Project Objective
To analyze restaurant sales data using SQL and extract business insights.

---

# Database Overview

Database Name: zomato_analytics

Tables:
- customers
- restaurants
- menu
- orders
- order_details

Relationships:
- Customers place Orders
- Restaurants have Menu items
- Orders contain multiple Order Details
- Order Details reference Menu items

---

# Business Insights Generated

1. Total Revenue Generated
2. Top Revenue-Generating Restaurants
3. Most Popular Food Item
4. Revenue Distribution by City
5. High-Value Customers

---

# SQL Concepts Used

- INNER JOIN
- GROUP BY
- ORDER BY
- Subqueries
- Aggregate Functions (SUM, AVG)
- Foreign Key Constraints

---

# Learning Outcome

- Database schema design
- Relational mapping
- Analytical query writing
- Business-driven SQL analysis
