# Zomato Analytics SQL Project

# Project Overview
This project analyzes food delivery data using SQL.

# Database Name
zomato_analytics

#Tables Created
- customers
- restaurants
- menu
- orders
- order_details

# Analytical Queries
- Total Revenue
- Top Restaurants by Revenue
- Most Ordered Food Item
- City-wise Revenue
- Customers Ordering Above Average

# Skills Used
- SQL Joins
- Aggregate Functions
- Subqueries
- Group By
- Data Analysis
